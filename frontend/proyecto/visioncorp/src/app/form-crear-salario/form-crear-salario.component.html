El botón para crear el registro de salario se habilitará una vez que todos los campos del formulario se completen correctamente.

Datos de registro de sueldos

Por favor, completa correctamente todos los campos del formulario.
El salario diario es obligatorio
El salario base debe ser un número decimal con dos dígitos después del punto.
El aguinaldo es obligatorio
El aguinaldo debe ser un número entero.
La Prima Vacacional es obligatoria.
La Prima Vacacional debe ser un número decimal de 2 dígitos entre 0 y 1.
Las Vacaciones es obligatorio
Las Vacaciones debe ser un número entero.
La fecha de actualización es obligatoria.
